# 19 — 单变量与多输入反向测算

**What to build:** 在中间卡片工作区交付单变量反向和多输入反推。变量卡只配置搜索范围、初始值和优先级；约束卡配置指标、明确年份、目标值、`=`/`≥`/`≤` 关系及硬/软属性。单变量模式允许一张变量卡，多输入模式允许多张变量卡，同一优先级可共同搜索。结果按硬约束、软偏差、变量数量、基准偏离和优先级排序，并解释变量启用与调整原因。反向始终手动执行。

**Blocked by:** 17 — 指标导航、卡片编辑与展示偏好; 18 — 正向测算与 C–J 结果表

**Status:** ready-for-human

- [x] 单变量模式可配置一张变量卡和多张约束卡并手动求解
- [x] 多输入模式支持多变量卡、优先级、同级变量叠加和手动求解
- [x] 每条约束绑定一个明确预测年份，默认 2030
- [x] 关系支持 `=`、`≥`、`≤`，并区分硬约束与软目标
- [x] 无可行解、硬约束命中/未命中和软目标偏差清晰呈现
- [x] 可行解按 PRD 规定顺序排序并展示搜索摘要、启用变量和调整幅度
- [x] 反向结果可保存为场景并回写左右导航状态
