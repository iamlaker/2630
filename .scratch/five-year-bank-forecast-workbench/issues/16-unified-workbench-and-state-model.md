# 16 — 统一四模块工作台与状态模型

**What to build:** 将工作台重构为统一的四页签外壳：正向单场景、单变量反向、多输入反推、规则集维护。四个模块共享活动模板、场景和计算引擎，但分别保留各自的指标选择、卡片排序、输入编辑、变量和约束草稿；规则集维护不继承测算草稿。三栏宽度可拖动、双击恢复，并按模块保存在当前浏览器；窄屏切换为抽屉式布局。

**Blocked by:** None — can start immediately

**Status:** ready-for-human

- [x] 四个模块在同一工作台顶部页签中可切换，且共享模板、场景、引擎上下文
- [x] 模块专属草稿在切换页签后保留，规则集维护与测算草稿隔离
- [x] 左中右三栏可拖动调整、双击恢复默认，并按模块记忆
- [x] 窄屏下布局可用，不出现不可操作的横向挤压
- [x] 页面刷新或切换模块不会丢失内存中的未提交状态
