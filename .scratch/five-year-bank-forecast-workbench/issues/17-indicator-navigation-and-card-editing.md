# 17 — 指标导航、卡片编辑与展示偏好

**What to build:** 在统一工作台中交付左右指标导航和中间卡片编辑体验。输入与输出按业务分组默认折叠；管理员默认展示项全局生效，用户星标在浏览器内跨模块共享。指标状态使用低饱和度并列标签。中间支持多选指标、每屏六张卡自动分页、排序和移除；正向卡片支持默认“年份纵排 + 横向滑杆”，并可切换为“年份横排 + 纵向滑杆”，同时保留精确输入、基准差异和年度联动。

**Blocked by:** 16 — 统一四模块工作台与状态模型

**Status:** ready-for-human

- [x] 左侧输入与右侧输出支持业务分组、折叠、搜索和命中自动展开
- [x] 管理员默认指标、用户星标和模块展示选择按约定范围保存
- [x] 已选、已修改、已设置约束、已有结果和异常状态可同时显示
- [x] 选中超过六个指标后自动生成第二个及后续卡片屏
- [x] 正向卡片提供两种布局，默认横向滑杆布局，切换不改变数据语义
- [x] 卡片支持恢复基准、精确输入、规则范围/步长和联动策略
