# 08 - 实现单变量反向测算 v1

**What to build:** 用户可以设置若干输入或输出约束，选择一个可变输入指标，系统通过反复调用正向 Excel 测算寻找满足目标的指标值，并展示可行性、约束命中情况、偏差和验证状态。

**Blocked by:** 06 - 保存和管理命名场景; 07 - 支持长任务进度与取消

**Status:** ready-for-human

- [x] 用户可以选择约束指标、约束类型、上下限或目标值，并单独决定是否启用每项约束。
- [x] v1 支持固定若干约束和一个选定变量指标，不要求多变量全局优化。
- [x] 系统返回可行解、变量所需值或调整幅度、每项约束命中/未命中及软约束偏差。
- [x] 无可行解、规则未确认、循环未收敛和计算失败均返回明确原因。
- [x] 用户能够将有效的反向测算结果保存为命名场景。
- [x] 反向测算允许进度展示和取消，并记录完整审计信息。
